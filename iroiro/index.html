IroIro is a small note-taking app that I built with a few people as part of an iOS programming class back in college. The goal was to build an app that would be as simple as possible.

Notes would be organized using color-coded tags. The tags could then be referenced within notes by mentioning the #tag inline.

The system was inspired by a problem I had with folder-based organization approaches: I'd always want to put a note into multiple buckets.
